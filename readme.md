## airtest-demo示例项目

> 该项目实现了脱离airtestIDE的脚本


### 运行项目

在项目的根路径

```shell
  sh run.sh
```

### 报告生成

- 可以生成IDE一样的airtest-report
- 可以通过pytest生成测试报告
- 可以生成allure报告
